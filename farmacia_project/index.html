Sobre el sistema

Este sistema permite gestionar los productos disponibles en la farmacia, incluyendo:

  • Agregar nuevos productos.
  • Editar la información de productos existentes.
  • Eliminar productos que ya no están disponibles.
  • Visualizar una lista completa de los productos registrados.
